# Тестовое задание: Интерфейс заявок

Разработка адаптивного интерфейса с использованием современного стека технологий.

## Задача

Реализовать верстку и базовую интерактивность экранов по предоставленному макету, с поддержкой адаптивности и элементами пользовательского взаимодействия.

## Стек технологий

- **React** + **Vite** (сборка и разработка)
- **Chakra UI** (библиотека компонентов)
- **TypeScript** (типизация)

> *Разрешается использовать любые дополнительные библиотеки на выбор*

## Основные требования

### 1. Верстка

- Реализовать десктопную и мобильную версии экранов
- Верстка должна быть адаптивной и отзывчивой
- Использовать компоненты **Chakra UI**

### 2. Интерактивность

- Добавить переключение по табам
- Реализовать создание заявки (через модальное окно)

#### В модальном окне:
- Реализовать выпадающие списки (Select) с моковыми данными
- Реализовать подгрузку фотографий

#### Логика:
- После нажатия на кнопку "Создать" — закрывать модалку
- Выводить введенные данные в консоль
